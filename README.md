## 🛠️ Tech Stack

- **HTML5 / CSS3 / Vanilla JS** — no frameworks, no build tools
- **Fonts** — Syne (headings) + JetBrains Mono (body)
- **Icons** — devicons via jsDelivr CDN
- **Animations** — IntersectionObserver scroll reveal, CSS keyframes
- **Hosting** — GitHub Pages
- **Built with** — [Claude Code](https://claude.ai/code)

## ✏️ Adding LinkedIn Posts

The LinkedIn section has a built-in template. Open `index.html`, find the comment `<!-- PASTE POSTS HERE -->`, and paste the card block with your real post data:

```html
<div class="li-card reveal">
  <div class="li-card-header">
    <div class="li-avatar">MM</div>
    <div>
      <div class="li-name">Muhammad Muneeb</div>
      <div class="li-date">March 2025</div>
    </div>
  </div>
  <div class="li-text">Your post text here...</div>
  <div class="li-stats">
    <div class="li-stat">♥ 42 likes</div>
    <div class="li-stat">💬 8 comments</div>
    <a href="YOUR_POST_URL" target="_blank">View post →</a>
  </div>
</div>
```
